## เริ่มต้น

ใส่รายชื่อ **dependencies** ในไฟล์ `package.json` และรันคำสั่ง `npm install`

```json
"dependencies": {
    "expo": "^32.0.0",
    "react": "16.5.0",
    "native-base": "^2.8.1",
    "@expo/vector-icons": "8.1.0",
    "react-native": "https://github.com/expo/react-native/archive/sdk-32.0.0.tar.gz"
  },
```
